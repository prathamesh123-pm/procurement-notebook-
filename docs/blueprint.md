# **App Name**: ProcurePal

## Core Features:

- Daily Task Log: A feature to create, update, and track daily tasks for team members, including status (assigned, pending, completed).
- Route Definition & Management: Add new procurement routes with details like name, current distance (km), assigned vehicle, and total number of suppliers, and update existing route information.
- Supplier Profile Management: Create and manage detailed profiles for each milk supplier, including name, address, mobile number, current milk quantity, milk quality, and other minimal information, with full editing capabilities.
- Supplier-Route Association: Tools to easily assign suppliers to specific routes, and manage these associations by adding or removing suppliers from a route.
- Visit Report Creator: Functionality to create and save visit reports, capturing observations and interactions during route visits for future reference.
- Daily Operations Report Compiler: A module to compile and save consolidated daily work reports, integrating task logs and other operational data.
- Smart Note Summarizer Tool: An AI tool to assist in summarizing raw text notes from supplier interactions or daily work observations into concise, actionable bullet points for reporting.

## Style Guidelines:

- Primary color: A clean and dependable mid-tone blue (#3F7FBF), representing trust and clarity.
- Background color: A very light, desaturated blue (#EEF1F4), providing a clean and expansive backdrop.
- Accent color: A deep, rich indigo-purple (#3D298C), offering strong contrast for interactive elements and highlights.
- Headlines: 'PT Sans', a humanist sans-serif for a modern feel with a touch of warmth. Body text: 'Inter', a neutral and highly readable sans-serif for clarity in data presentation.
- Utilize a consistent set of clean, outlined icons for actions, navigation, and status indicators, ensuring clarity and modern aesthetic.
- A dashboard-style layout with clear, logical sections for task management, route details, and supplier profiles. Employ responsive grids for optimal viewing on various devices, ensuring forms and data tables are well-structured and easy to parse.
- Subtle, smooth transitions for page loads and component changes, and delicate hover effects on interactive elements to provide user feedback without distraction.
